Coding Platforms

1.) HackerRank

HackerRank is a very good website to start practicing coding. There are tones of questions to practice. Questions are divided in various difficulty levels. To start with, one should first solve the easy level questions and then proceed for higher difficulty level questions. There can also be contests organised on HackerRank, but these contests are not open to all, only a private room can be created to organise a contests/competition among competitors.

2.) Codeforces

Codeforces is best site for sharping ones coding skills. It is also currently most active websites all around world. It organises a contests almost every weak. The contests are divided in various difficulty levels namely, Division 1, Division 2, Division 3 with toughest is Div. 1 and easiest is Div. 3. These contests are organised at global level, participents are over the world. There is also a rating system in it, in which every ones rating at start is 1500, but as more contests are given, the rating either improves or decreases depending on ones performances.

3.) Codechef

CodeChef is a competitive programming website. It is a non-profit educational initiative of Bhavin Turakhia, aimed at providing a platform for students, young software professionals to practice, hone their programming skills through online contests Apart from this, the 'CodeChef For Schools' program aims to reach out to young students and inculcate a culture of programming in Indian schools.

4.) Stack Overflow

Stack Overflow is a question and answer site for professional and enthusiast programmers. It is a privately held website. It features questions and answers on a wide range of topics in computer programming.It was created to be a more open alternative to earlier question and answer sites such as Experts-Exchange.
